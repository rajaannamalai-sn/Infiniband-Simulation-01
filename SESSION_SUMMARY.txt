KEY LEARNING: TOPOLOGY FORMAT
------------------------------
The critical requirement was using the EXACT ibnetdiscover(8) format:

CORRECT FORMAT (Working):
  vendid=0x2c9
  devid=0x5a44
  caguid=0x0002c90200200002
  Ca 2 "H-0002c90200200002"
  [1](2c90200200003) "S-0002c90200200001"[1]

INCORRECT FORMAT (Failed):
  Hca 1 "H-0x0002c902002002" 0x0002c902002002
  0x0002c902002002 1 0x0002c902002001 1

Key differences:
- Use "Ca" not "Hca"
- Include vendid, devid, caguid before each node
- Port connections need GUID in parentheses: [port](guid)
- Bidirectional connections required (both switch→host and host→switch)
